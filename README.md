# 🍅 番茄时钟

一个简洁的番茄工作法计时器，支持任务分类、云端同步和历史记录。

## ✨ 功能

- **番茄计时** — 25 分钟专注 + 5 分钟休息，经典番茄工作法
- **刷新不丢** — 计时状态多重持久化，刷新页面计时继续
- **任务分类** — 自定义专注类型（学习、工作、阅读…）
- **备注记录** — 每个番茄钟可添加备注，可记录具体专注事项
- **日历视图** — 按月查看每日专注次数，分早中晚时段统计
- **云端同步** — 通过注册后登录账号，多设备同步专注数据（不登录也可使用只是不同步专注数据）

## 🛠️ 技术栈

- 原生 HTML / CSS / JavaScript
- [Supabase](https://supabase.com) — 用户认证与数据同步
- 无需构建工具，开箱即用

## 🚀 部署

静态站点，可直接部署到 [Netlify](https://netlify.com)、Vercel 或任意静态托管服务。
